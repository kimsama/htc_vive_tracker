### Instalation instructions

* Install dependencies (OpenVR)

Download and install openvr from this [link] (https://github.com/ValveSoftware/openvr)


```

```

* Create build folder

```
cd htc_vive_tracker
mkdir build
cd build
```

* Compile and install library


```
cmake ..
make
sudo make install
```
